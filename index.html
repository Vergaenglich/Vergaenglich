# Vergänglich — Photography Portfolio ## Adding new photos 1. Drop the image file into the `/photos/` folder 2. Open `photos.js` in a text editor 3. Add a new line inside the `PHOTOS` array: ```js { src: 'photos/YOUR_FILENAME.jpg', title: 'your title', caption: 'your caption.', category: 'street', city: 'Berlin' }, ``` 4. Save the file 5. Push to GitHub — Netlify deploys automatically ## Categories - `street` - `architecture` - `portrait` - `documentary` - `abstract` ## Deploying via GitHub + Netlify 1. Create a free account at github.com 2. Create a new repository called `vergaenglich` 3. Upload all files (index.html, photos.js, README.md, and the /photos/ folder) 4. Go to netlify.com → New site → Import from GitHub 5. Select your repository → Deploy Every time you add photos and push to GitHub, the site updates automatically. ## Contact Update the email address in two places in `index.html`: - Search for `hello@vergaenglich.photography` and replace with your real email (appears 3 times)
